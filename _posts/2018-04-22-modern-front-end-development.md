作为web前端开发的你，如果你现在仍然停留在使用jquery, dreamweaver 上， 你已经out很多年了。 随着个人电脑硬件计算能力的提高，现在浏览器已经可以较为快速的完成很多工作。之前的mvc开发模式也已经不在光鲜。现代前端开发，我们还需要了解很多新的设计思路，新的观点。
1. SPA - single page application
当前， web开发越来越复杂， 以前整业刷新的方式变得越来越难以满足需求。 基于jquery的ajax局部刷新方案也被证实无法满足富web程序的需求。 所以越来越多的工程师把controller层移到了前端来做，具体来说，就是web程序越来越像一个客户端软件。 使用标准API和后端进行交互。
2. ES6 - ECMAScript 6
Javascript发展到第六代了， 新的语法更加简洁有力，表现力极强。http://es6-features.org/#Constants
3. Node是运行在服务器上的js. 随着的发展越来越多的程序员喜欢使用node, node 的生态也已经发展为一个生态了。 
4. Yarn 是一款node平台下的包依赖管理软件。https://yarnpkg.com/en/
5. Lodash: A modern JavaScript utility library delivering modularity, performance, & extras. https://github.com/lodash/lodash
6. babel: 是一款现代的js编译器。https://babeljs.io/
7. grunt: https://gruntjs.com/
8. immutable-js: https://facebook.github.io/immutable-js/
